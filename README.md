# hello-grpc-go

minimal example of gRPC in Go.

## Requirements

* [Go 1.16](https://golang.org/)
* [protoc](https://developers.google.com/protocol-buffers)


## Compile proto file

```bash
./compile_proto.sh
```

## Build

### Server

```bash
cd server
go build
```

### Client

```bash
cd server
go build
```
